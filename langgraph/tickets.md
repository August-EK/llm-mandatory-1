Title: Create Basic To-Do List REST API
Scope: Implementation of a basic Flask To-Do List REST API with endpoints for retrieving all To-Do items, creating new ones, retrieving specific items by ID, updating items by ID, and deleting items by ID.
Definition of Done:

* All required endpoints are implemented in the `routes.py` file.
* The `models.py` file defines the necessary database models for To-Do items.
* The `schemas.py` file includes serialization/deserialization schema definitions for To-Do items.
* The `app.py` file contains the main application logic and setup for the Flask app, including the routes and configuration settings from the `config.py` file.
